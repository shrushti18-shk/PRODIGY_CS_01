## üìå What is Caesar Cipher?

Caesar Cipher is a **simple substitution encryption technique** where each letter in the message is shifted by a fixed number of positions in the alphabet.

### Example:
- Message: `HELLO`
- Shift: `3`
- Encrypted: `KHOOR`

---

## üöÄ Features

- Encrypts text using a user-defined shift value
- Decrypts the encrypted text back to the original message
- Supports both **uppercase and lowercase letters**
- Keeps spaces and special characters unchanged
